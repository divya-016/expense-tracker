# Expense Tracker 💰
### Browser-Based Web Application

A simple expense tracker that runs entirely in the browser 
— no installation or backend needed. Just open the HTML 
file and start tracking!

## Features
- Add expenses with date, category, amount and description
- Live summary cards — total spend, top category, 
  this month's total
- Doughnut chart showing spending breakdown by category
- Filter expenses by category
- Export all data as CSV to open in Excel
- Data saved in browser localStorage — stays even after 
  closing the tab

## Categories
Travel, Client Entertainment, Office Supplies, Software, 
Meals, Audit Fees, Miscellaneous

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Chart.js
